# Portfolio - Daniel Isaac Elgueta Fuentes

Portafolio profesional de Ingeniería en Informática con enfoque en Data Analytics, Business Intelligence y Machine Learning.

## Tecnologías

- **Frontend**: React.js, CSS3, Bootstrap
- **Herramientas**: Git, Visual Studio Code
- **Datos**: Python (pandas, numpy), SQL Server, Power BI

## Estructura del Proyecto

```
Portafolio/
├── public/                 # Archivos públicos
├── src/
│   ├── Assets/            # Recursos (imágenes, CVs)
│   ├── components/        # Componentes React
│   │   ├── About/
│   │   ├── Home/
│   │   ├── Projects/
│   │   └── Resume/
│   ├── App.js
│   └── index.js
├── CV_Engineering_Classic_ES.pdf   # CV Profesional
├── CV_Engineering_Classic_ES.yaml   # CV Fuente (YAML)
└── package.json
```

## Instalación

```bash
npm install
npm start
```

La aplicación estará disponible en `http://localhost:3000`
